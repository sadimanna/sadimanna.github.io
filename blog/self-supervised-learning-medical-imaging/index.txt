e:T1257c,<h2 id="introduction"><a class="heading-anchor" aria-hidden tabindex="-1" href="#introduction"><span class="anchor-icon">#</span></a>Introduction</h2>
<p>Medical image analysis has been transformed by deep learning, yet a persistent bottleneck remains: the <strong>scarcity of expert annotations</strong>. Labeling medical images requires domain expertise (radiologists, pathologists), making large-scale annotation prohibitively expensive.</p>
<p>Self-Supervised Learning (SSL) offers a compelling solution by learning meaningful representations from <strong>unlabeled data</strong> — of which medical institutions have vast amounts.</p>

                 bottleneck_dim: <span class="hljs-built_in">int</span> = <span class="hljs-number">256</span>, out_dim: <span class="hljs-built_in">int</span> = <span class="hljs-number">65536</span></span>):
        <span class="hljs-built_in">super</span>().__init__()
        <span class="hljs-variable language_">self</span>.mlp = nn.Sequential(
            nn.Linear(in_dim, hidden_dim),
            nn.GELU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.GELU(),
            nn.Linear(hidden_dim, bottleneck_dim),
        )
        <span class="hljs-variable language_">self</span>.last_layer = nn.utils.weight_norm(
            nn.Linear(bottleneck_dim, out_dim, bias=<span class="hljs-literal">False</span>)
        )
        <span class="hljs-variable language_">self</span>.last_layer.weight_g.data.fill_(<span class="hljs-number">1</span>)
    
    <span class="hljs-keyword">def</span> <span class="hljs-title function_">forward</span>(<span class="hljs-params">self, x: torch.Tensor</span>) -> torch.Tensor:
        x = <span class="hljs-variable language_">self</span>.mlp(x)
        x = F.normalize(x, dim=-<span class="hljs-number">1</span>, p=<span class="hljs-number">2</span>)
        x = <span class="hljs-variable language_">self</span>.last_layer(x)
        <span class="hljs-keyword">return</span> x


<span class="hljs-keyword">class</span> <span class="hljs-title class_">DINOLoss</span>(nn.Module):
    <span class="hljs-string">"""Cross-entropy loss for DINO self-distillation."""</span>
    
    <span class="hljs-keyword">def</span> <span class="hljs-title function_">__init__</span>(<span class="hljs-params">self, out_dim: <span class="hljs-built_in">int</span>, teacher_temp: <span class="hljs-built_in">float</span> = <span class="hljs-number">0.04</span>,
                 student_temp: <span class="hljs-built_in">float</span> = <span class="hljs-number">0.1</span>, center_momentum: <span class="hljs-built_in">float</span> = <span class="hljs-number">0.9</span></span>):
        <span class="hljs-built_in">super</span>().__init__()
        <span class="hljs-variable language_">self</span>.teacher_temp = teacher_temp
        <span class="hljs-variable language_">self</span>.student_temp = student_temp
        <span class="hljs-variable language_">self</span>.center_momentum = center_momentum
        <span class="hljs-variable language_">self</span>.register_buffer(<span class="hljs-string">"center"</span>, torch.zeros(<span class="hljs-number">1</span>, out_dim))
    
    <span class="hljs-keyword">def</span> <span class="hljs-title function_">forward</span>(<span class="hljs-params">self, student_output: torch.Tensor,
                teacher_output: torch.Tensor</span>) -> torch.Tensor:
        student_probs = F.log_softmax(
            student_output / <span class="hljs-variable language_">self</span>.student_temp, dim=-<span class="hljs-number">1</span>
        )
        teacher_probs = F.softmax(
            (teacher_output - <span class="hljs-variable language_">self</span>.center) / <span class="hljs-variable language_">self</span>.teacher_temp, 
            dim=-<span class="hljs-number">1</span>
        ).detach()
        
        loss = -torch.<span class="hljs-built_in">sum</span>(
            teacher_probs * student_probs, dim=-<span class="hljs-number">1</span>
        ).mean()
        
        <span class="hljs-comment"># Update center with EMA</span>
        <span class="hljs-variable language_">self</span>.center = (
            <span class="hljs-variable language_">self</span>.center_momentum * <span class="hljs-variable language_">self</span>.center + 
            (<span class="hljs-number">1</span> - <span class="hljs-variable language_">self</span>.center_momentum) * teacher_output.mean(dim=<span class="hljs-number">0</span>)
        )
        
        <span class="hljs-keyword">return</span> loss

<h2 id="conclusion"><a class="heading-anchor" aria-hidden tabindex="-1" href="#conclusion"><span class="anchor-icon">#</span></a>Conclusion</h2>
<p>Self-supervised learning is rapidly closing the gap with supervised methods in medical imaging, and in some cases surpassing them — especially in low-data regimes. Key takeaways:</p>
<ul>
<li><strong>Domain-specific augmentations</strong> are critical for medical SSL</li>
<li><strong>Vision Transformers</strong> with self-distillation (DINO) produce semantically rich features</li>
<li><strong>Foundation models</strong> trained on paired image-text data enable zero-shot generalization</li>
<li>The path forward combines <strong>SSL pretraining</strong> with <strong>efficient fine-tuning</strong> (LoRA, adapters) for clinical deployment</li>
</ul>
